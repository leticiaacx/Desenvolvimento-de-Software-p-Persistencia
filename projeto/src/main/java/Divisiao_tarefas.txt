1. Para o projeto que acompanhará todo o curso, nós, eu e minha dupla Giovanna Moncayo, optamos por criar uma aplicação relacionada uma agenda de Contatos.
Nosso objetivo é persistir dados sobre os contatos em uma lista com varios outros contatos. 
Por enquanto, estamos limitando as funcionalidades da aplicação às que foram requisitadas para a primeira etapa do trabalho

Definir uma Entidade e Criar uma Classe Java: Escolhemos a esntidade "Contato". 
A entidade "Contato" é composta pelos seguintes atributos:

Nome
Sobrenome
Email
Endereço
Numero
Aniversario

2. Criar uma Classe Principal com um Menu: Implementamos uma classe principal que contém um menu com as seguintes funcionalidades:

F1. Inserir Entidade no Arquivo CSV: Permite cadastrar dados relacionados à entidade "Contato" digitados via teclado e adicioná-los a um arquivo CSV.

F2. Mostrar a Quantidade de Entidades no Arquivo CSV: Mesmo que a aplicação seja reiniciada, esta funcionalidade sempre mostrará a quantidade real de contatos no arquivo CSV, mesmo que os dados tenham sido inseridos ou removidos externamente através de um editor de texto.

F3. Converter Dados para JSON e XML: Utilizamos a biblioteca Jackson para converter os dados do arquivo CSV em arquivos JSON e XML.

F5. Compactar o Arquivo CSV: Utilizamos a API Java para comprimir o arquivo CSV em um arquivo ZIP com o mesmo nome.

F6. Mostrar o Hash SHA256 do Arquivo CSV: Utilizamos a API Java para calcular e exibir o hash SHA256 do arquivo CSV.
